# TwinkoTalk Tarot Twinko Expression & Usage Specification

**Version:** v3.0  
**Status:** founder-reviewed character usage spec

## Canonical foundation
All Tarot Twinko assets must preserve:
- founder-approved yellow star silhouette
- black eyes with highlights
- orange cheeks
- purple wizard hat with yellow star ornament
- purple cape with gold clasp
- no separate arms or legs
- soft illustrated dimensionality
- transparent background

## Approved state set

### Idle
- filename: `twinko_tarot_idle_v1.png`
- use: landing, topic, question, spread selection, waiting
- emotion: welcoming, calm

### Interpreting
- filename: `twinko_tarot_interpreting_v1.png`
- use: interpretation loading, reflective guidance moments
- emotion: thoughtful, reassuring

### Magic / Ritual
- filename: `twinko_tarot_magic_v1.png`
- use: shuffle ritual, magical transition
- prototype may keep baked cards / glow / sparkles
- future production can separate these into code layers

### Summary / Completion
- filename: `twinko_tarot_summary_v1.png`
- use: final summary, guidance summary card, share / save completion state
- emotion: warm, complete, softly celebratory

## Hard rules
- do not redraw Twinko in a different material style
- do not add separate arms or legs
- do not substitute random unofficial Tarot Twinko poses
- do not change the approved hat / cape family
